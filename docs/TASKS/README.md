## Task Template

## Workflow

1. Create task in `TODO/` folder
2. Move to `IN_PROGRESS/` when starting
3. Update task with findings and changes
4. Move to `COMPLETED/` when done
5. If blocked, move to `BLOCKED/` with explanation
